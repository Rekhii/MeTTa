# MeTTa Programming

Learning MeTTa programming language - weekly assignments for iCog Labs internship.

## Week 1 - Recursive Lists & Peano Numbers

### Recursive Lists
Implemented a recursively defined list type using `Cons` and `Nil`, along with:
- **is_member** - checks if a value exists in the list
- **prepend** - adds a value to the front of the list
- **append** - adds a value to the end of the list
- **length** - counts the number of elements in the list

### Peano Numbers
Implemented Peano number representation using `Zero` and `Succ`, along with:
- **to-peano** - converts Arabic numerals to Peano numbers
- **from-peano** - converts Peano numbers back to Arabic numerals
- **peano-add** - addition over Peano numbers
- **peano-sub** - subtraction over Peano numbers (floors at Zero)

### How to run
```bash
metta Week-1/MeTTa_Recursive_Lists/Recursive_Lists.metta
metta Week-1/MeTTa_Peano_Numbers/Peano_Numbers.metta
```
